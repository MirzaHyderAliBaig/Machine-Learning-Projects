# Halifax Tax Rate Prediction

This project predicts municipal tax rates in Halifax using linear regression.

## Dataset
- Source: Halifax Open Data (https://data-hrm.hub.arcgis.com/datasets/a1d6cb118da642a4ae6a7b3191ae2369_0/explore)
- Features: Bill_Year, Minimum_Rate, Maximum_Rate, Calculation_Type, Rate_Type, Rate_Code
- Target: Rate

## Steps
1. Data cleaning & preprocessing
2. Feature encoding (categorical variables)
3. Log transformation & outlier removal
4. Train/test split & scaling
5. Train Linear Regression model
6. Evaluate performance (MSE, RÂ²)
7. Visualize predicted vs actual rates
